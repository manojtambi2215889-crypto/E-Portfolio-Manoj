# Business Portfolio Website

Welcome to the **Business Portfolio** static website. This project is a premium‑looking personal portfolio built with vanilla HTML, CSS, and JavaScript, designed for seamless deployment on Vercel (or any static‑hosting platform).

## ✨ Features
- Modern glassmorphism UI with vibrant gradients and dark/light mode toggle.
- Responsive layout that works across desktop, tablet, and mobile devices.
- Interactive sections (skills, projects, certificates) powered by plain JavaScript.
- Clean, semantic markup for SEO friendliness (proper `<title>`, meta description, heading hierarchy, etc.).
- No build step required – just static files.

## 🛠️ Tech Stack
- **HTML5** – semantic structure, SEO‑optimized headings.
- **CSS3** – custom design system, CSS variables for theming, glass‑like cards, smooth transitions.
- **JavaScript** – theme toggling, simple UI interactions.
- **Google Fonts** – modern typography (e.g., Inter).
- **Lucide Icons** – lightweight SVG icon set via CDN.

## 🚀 Deployment
The site is fully compatible with **Vercel** (zero‑config). To deploy:
1. Push the repository to GitHub (or any Git provider).
2. Import the repo into Vercel.
3. Vercel will automatically detect the static site and serve it.

If you prefer a local preview:
```bash
# Clone the repo (if not already local)
git clone <repo-url>
cd business-portfolio
# Open index.html in a browser (or use a simple server)
start index.html   # Windows
# or
python -m http.server 8000   # then visit http://localhost:8000
```

## 📄 Assets & Certificates
The README references two certificates stored locally. For a production deployment, move the PDF files into an `assets/` folder inside the project and update the links in **index.html** accordingly.
